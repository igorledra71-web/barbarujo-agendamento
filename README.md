# barbarujo-agendamento

App web de agendamento para barbearia com Firebase Realtime Database.

## Como usar (modo simples)

Se você quiser editar de forma prática em **um único arquivo**, use só o `index.html`.
Ele já contém:

- HTML da interface
- CSS inline
- JavaScript completo (incluindo Firebase)

Ou seja: dá para apagar/colar o conteúdo inteiro do `index.html` sem depender de `app.js` e `styles.css`.

## Arquivos auxiliares

- `manifest.json`: configuração PWA
- `sw.js`: service worker
- `icon-192.png` e `icon-512.png`: ícones do app
- `logo.png`: logo da splash/topo
